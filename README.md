# MidiSense

Software for MidiSense, a modular system for gestural sound design developed by Stefanos Skialivas and Nicola Privato.

App/MidiApp - This is a Max Standalone to interface the Midisense bracelets to any DAW or sofware supporting MIDI mapping. To download the app: https://drive.google.com/drive/folders/1ERg0Lt57HHWPwC5U1LfL9h0Eu1uePiWs?usp=sharing

To run, simply download the app, unzip, plug the Midisense receiver, open the app and select MidiSense as MIDI input. On the DAW side, select MidiApp as input device.

ESP32 - includes receiver and sender code

3D_models - 3D print files to build one MidiSense device

RAVE - features a RAVE Decoder trained on organ sounds with vinyl crackles, with 15 latents cmapped 1 to 1 to the Midisense gyros xyz dimensions
